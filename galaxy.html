Individual tools

How to get the individual Galaxy-GSA tools:

If you have your own Galaxy instance, it is easy to download the individual tools we have generated. Go to the Toolshed and search for the user "mora-lab". Then, select and download the desired tools. If you need help to install your own Galaxy instance and get tools from the toolshed: 1. Installing Galaxy in a VM
